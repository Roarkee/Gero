# Gero
Freelancer Client Management App


Gero is a full-stack web application designed to help individuals and businesses manage their freelance clients, projects, tasks, and subtasks more efficiently. It provides a streamlined dashboard for organizing work, tracking progress, and collaborating effectively.

---

## ✨ Features

### 🔒 Authentication
- Secure login and registration (JWT-based)
- Role-based access for users

### 🧑‍💼 Client Management
- Add, update, and delete clients
- View client details and history

### 📁 Project Management
- Link projects to specific clients
- Assign statuses, deadlines, and progress

### ✅ Task & Subtask Tracking
- Nested task structures
- Track task status, priority, and timelines

### 🏷️ Labels & Tags
- Organize tasks with custom labels

---

## 🛠️ Tech Stack

### Backend
- **Django** + **Django REST Framework**
- PostgreSQL
- JWT Authentication

### Frontend
- **React** with **Vite**
- TailwindCSS or CSS Modules

### Dev Tools
- Docker (optional)
- Git/GitHub for version control
